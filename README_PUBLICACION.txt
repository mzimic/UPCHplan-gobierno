Publicación en GitHub Pages

1. Descomprimir este ZIP.
2. Subir todo el contenido al repositorio de GitHub (raíz del repositorio).
3. Verificar que en la raíz queden:
   - index.html
   - styles.css
   - .nojekyll
   - carpeta docs/
   - carpeta anexos/
4. En GitHub: Settings -> Pages.
5. En Build and deployment:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
6. Guardar.
7. Esperar unos minutos y abrir la URL pública.

Este paquete recupera la estructura narrativa de la versión web inicial y la adapta a la nueva organización de 4 anexos.
